# Docs as Code Platform

Welcome to the internal Documentation Portal for the Enterprise!

Our platform embraces the **Docs-as-Code** philosophy. By treating documentation like software, engineering teams can version, review, and deploy technical knowledge alongside the systems they build safely within standard version control practices.

## Key Features

- **Unified Discovery Portal:** A single entry point aggregates federated Markdown repositories into a sleek, central catalog.
- **Diagrams-as-Code:** Leverage standard Mermaid syntax to version and logically render rich architectural diagrams.
- **Content Tabs:** Use MkDocs-compatible `=== "Tab Title"` syntax to create switchable code examples and multi-platform guides.
- **Syntax Highlighting:** Server-side code highlighting powered by Shiki with full theme support.
- **Local Authoring Preview:** Authors can use the custom CLI tools to preview their exact Markdown payload securely on their own machine before opening a Pull Request.
- **Global Search:** Instantly discover the context you need across the thousands of platform documents via our lightning-fast, privacy-first static search engine (⌘K).
- **Modular Architecture:** Built on Nx with shared Angular libraries — feature code is reusable across the portal and preview tools.
